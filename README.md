# ReddiHelp Marketing Website

Static one-page website for presenting the ReddiHelp Emergency Coordination Platform to non-technical viewers.

## Open locally

Open `index.html` in a browser. No build step or server is required.

## Contents

- `index.html` - page structure and marketing copy
- `styles.css` - responsive layout and visual design
- `script.js` - mobile navigation, role tabs, and reveal animations
- `assets/screenshots/` - product visuals extracted from the project report
